# Nützlichen Komponenten

im Aufbau... Aber basically:

- `RWTH/Main`: Layout
- `BasicSubjectInfo`: Links zu üblichen Sachen für ein Fach. E.g. panikzettel, folien-merge, ...
- `SubjectDocumentList`: Tree view aller Dokumente für ein Fach unter `/public/content-assets/[subject]`
- `SubjectList`: Auswahl zwischen mehreren Fächern.
- `SummaryList`: Listet alle Zusammenfassungen des Fachs
- `SummaryView`: Zeigt eine Zusammenfassung an.
